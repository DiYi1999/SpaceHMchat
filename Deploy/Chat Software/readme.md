### Special thanks to the developers of Cherry Studio for creating such an excellent open-source chat software!

### In this project, we use Cherry Studio as the frontend (client installation) chat software:
1. Our code was developed in 2025 and currently guarantees compatibility with Cherry Studio v1.6.2. Previous versions may be supported, but later versions might not be compatible due to apparent changes in the client interaction mode.
2. For Windows systems, you can directly extract and install the Cherry Studio v1.6.2 installation package from this folder. For other systems, please visit [https://github.com/CherryHQ/cherry-studio/releases](https://github.com/CherryHQ/cherry-studio/releases) to download the corresponding installation package for your system.

### Benefits of using this open-source chat software as the frontend:
1. Its high degree of commercialization means the knowledge base search and web search features have been market-validated, making them more stable and efficient than the RAG codes we wrote ourselves.
2. Open-source software provides code transparency, enhancing security and trustworthiness, especially suitable for fields with high information security requirements, such as aerospace and healthcare.
3. User-friendly interface, easy to use, and feature-complete. Using them directly is more professional and efficient than reinventing the wheel by developing chat software individually.
4. Open-source software can be modified and distributed to adapt to different needs, with strong community support for quick access to help and updates.

### Important chat software configuration parameters to note:
1. It is recommended to set the "Context" for the SpaceHMchat assistant as high as possible, with 50 being optimal and 20 being the minimum, to fully utilize conversation history information and improve response quality.
